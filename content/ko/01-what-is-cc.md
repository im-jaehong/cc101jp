# 01. Claude Code란?

> 내 컴퓨터 파일을 직접 읽고, 정리하고, 결과물로 만들어주는 AI 에이전트. 코딩을 몰라도 됩니다.

---

## 한 줄 정의

**Claude Code**는 내 컴퓨터 폴더에 직접 접근해서 일하는 AI입니다. 수십 개의 파일을 한 번에 읽고, 내용을 정리하고, 원하는 결과물을 만들어냅니다.

웹 브라우저에서 대화하는 Claude.ai와 달리, Claude Code는 **내 컴퓨터 안에서** 직접 작업합니다.

---

## Claude.ai 웹과 뭐가 다른가요?

| | Claude.ai (웹 브라우저) | Claude Code |
|---|---|---|
| **파일 접근** | 직접 올려야 함 (1개씩) | 폴더 전체 자동으로 읽음 |
| **작업 범위** | 대화 1회 분량 | 수십~수백 개 파일 동시 처리 |
| **결과물 저장** | 직접 복사해서 붙여넣기 | 파일로 자동 저장 |
| **반복 작업** | 매번 수동 | 한 번 말하면 알아서 반복 |
| **기억** | 대화 내에서만 | CLAUDE.md로 내 규칙 영구 기억 |

**핵심 차이**: Claude.ai는 대화창에서 답을 보여줍니다. Claude Code는 내 폴더에 직접 들어와서 결과물을 만들어 놓습니다.

---

## 어떻게 작동하나요? 실제 예시

**예시 1: 문서 일괄 정리 (마케터 / 기획자)**

```
"Downloads 폴더에 있는 회의록 30개를 읽고,
 날짜순으로 정리해서 이번 달 결정사항만 뽑아줘.
 월간_결정사항.md 파일로 저장해줘"

→ Claude Code가 하는 일:
1. 폴더의 파일 30개 전부 읽기
2. 날짜 파악 및 정렬
3. 결정사항 항목만 추출
4. 새 파일로 자동 저장
```

**예시 2: 데이터 분석 (연구자 / 분석가)**

```
"data 폴더에 있는 설문 결과 CSV 파일 분석해서
 주요 지표 요약 보고서 만들어줘"

파이썬 몰라도 됩니다. 말로 설명하면 알아서 처리합니다.
```

**예시 3: 코드 개발 (개발자)**

```
"테스트가 실패하는데 원인 찾아서 고쳐줘"
→ 테스트 실행 → 오류 파악 → 코드 수정 → 재실행
```

---

## 어떤 사람에게 맞는가?

### 이런 분에게 딱 맞습니다

| 상황 | 예시 |
|------|------|
| **반복적인 파일·문서 작업** | 매주 같은 형식의 보고서 정리, 대량 파일 이름 변경, 폴더 구조 통일 |
| **방대한 자료 분석** | PDF 논문 20개 핵심 요약, 경쟁사 웹페이지 일괄 수집·정리 |
| **아이디어를 실물로** | 코딩 없이 나만의 웹사이트, 자동화 도구, 대시보드 만들기 |
| **개발 작업** | 코드 작성, 버그 수정, 테스트 자동화 |

### 이런 분에게도 추천

- "파이썬이나 엑셀 함수 없이도 데이터 정리하고 싶다"
- "문서 수십 개를 일일이 열지 않고 한 번에 처리하고 싶다"
- 터미널이 낯설어도 괜찮습니다 — **데스크탑 앱**으로도 사용 가능

### 이런 분에게는 Claude.ai 웹이 더 적합

- 단순 글쓰기, 번역, 질문-답변만 필요한 분
- 특정 IDE 안에서만 작업하고 싶은 개발자 → Cursor나 VS Code 확장 고려

---

## 어디서 실행하나요?

| 환경 | 설명 |
|------|------|
| **데스크탑 앱** | GUI 인터페이스, 터미널 불필요 — **입문자 추천** |
| **터미널 (CLI)** | 가장 강력한 기본 환경 |
| **VS Code / Cursor 확장** | 에디터 안에서 바로 사용 |
| **웹 브라우저** | claude.ai/code 에서 바로 실행 |
