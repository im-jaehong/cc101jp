# 04. 製品ラインナップ＆価格

> Claude Codeを使う方法は3つあります。入門者ならMax $100プランから始めましょう。

---

## Claude Codeを使う3つの方法

### 方法1. Claude.aiサブスクリプション（Pro / Max）— 個人の入門者におすすめ

最もシンプルな方法です。Claude.aiでサブスクリプションに登録すれば、Claude Codeをすぐに使えます。トークン（使用量）をいちいち気にする必要なく、定額の月額料金を支払うだけです。

### 方法2. Anthropic APIの直接利用 — 開発者 / 自動化目的

Anthropic ConsoleでAPIキーを発行して接続する方式です。使用した分だけトークン（文字数単位）で課金されます。自動化パイプライン、CI/CD連携などに適しています。

### 方法3. エンタープライズクラウド — 企業のセキュリティ要件があるチーム

Amazon Bedrock、Google Vertex AI、Microsoft Foundryなどの企業向けクラウドを通じてClaude Codeを使用します。データが自社のクラウド内に留まります。

## Pro vs Maxプラン比較

> 以下の価格は2025年の公式基準です。最新の価格は[claude.com/pricing](https://claude.com/pricing)でご確認ください。

| | **Pro** | **Max（5x）** | **Max（20x）** |
|---|---|---|---|
| **月額** | $20 | $100 | $200 |
| **使用量の上限** | 基本 | Proの5倍 | Proの20倍 |
| **Claude Code含む** | 含む | 含む | 含む |
| **Claude.ai Web含む** | 含む | 含む | 含む |
| **適したユーザー** | たまに使用、お試し目的 | Claude Codeメインユーザー | ヘビーユーザー、大規模プロジェクト |

### 入門者におすすめ: Max $100

Claude Codeを本格的に使うなら**Max $100プラン**をおすすめします。その理由:

- Pro（$20）は使用量の上限にすぐ達し、Claude Codeを集中的に使うのが難しいです。
- APIの直接利用はトークン課金方式のため、予想外のコストが発生する可能性があります。
- Max $100は1日あたりの平均使用量を基準にすると、ほとんどの個人開発者に十分です。

公式ドキュメントによると、APIユーザーの平均コストは**開発者1人あたり1日約$6、月$100〜$200**程度です（使用頻度により差があります）。

## コスト構造の説明

### サブスクリプションプラン（Pro / Max）

```
月額決済 → 決められた使用量の上限内で自由に使用
上限超過時 → 追加の応答制限（追加課金なし）
```

- トークン数を気にする必要なし
- 上限に達した場合、一定時間後にリセット

### APIの直接利用

```
使用したトークン分だけ課金（入力トークン + 出力トークン）
モデルによって単価が異なる（Sonnet < Opus）
```

- `/cost`コマンドで現在のセッションのコストを確認可能
- 自動化/チーム利用時はConsoleで支出上限の設定が可能

---

## ⚠️ コストに関する注意事項

### API利用時の注意点

```
⚠️ API利用は使用した分だけ請求されます。
```

- 大規模プロジェクトで長時間作業するとトークンが急速に消費されます。
- Agent Team（複数のAIを同時実行）機能はトークン消費が約**7倍**多くなります。
- バックグラウンド作業（会話の要約など）にも少量のトークンが使用されます。

### コスト削減のヒント

| 方法 | 説明 |
|------|------|
| 作業の合間に`/clear`を実行 | 以前の会話コンテキストをクリアしてトークンを節約 |
| Sonnetモデルを優先使用 | Opusより安価で、ほとんどの作業に十分 |
| 明確なリクエスト | 「このコードベース全体を改善して」の代わりに特定のファイル/関数を指定 |
| MCPサーバーを最小化 | 使っていない接続は無効化 |
