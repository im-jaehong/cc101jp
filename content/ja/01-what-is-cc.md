# 01. Claude Codeとは？

> パソコンのファイルを直接読み込み、整理し、成果物として仕上げてくれるAIエージェント。プログラミングの知識は不要です。

---

## ひとことで言うと

**Claude Code**は、パソコンのフォルダに直接アクセスして作業するAIです。数十個のファイルをまとめて読み込み、内容を整理し、求める成果物を作成します。

ブラウザ上で会話するClaude.aiとは異なり、Claude Codeは**パソコンの中で**直接作業します。

---

## Claude.ai（Web版）との違い

| | Claude.ai（Webブラウザ） | Claude Code |
|---|---|---|
| **ファイルアクセス** | 手動でアップロード（1つずつ） | フォルダ全体を自動で読み込み |
| **作業範囲** | 1回の会話分 | 数十〜数百ファイルを同時処理 |
| **成果物の保存** | 手動でコピー＆ペースト | ファイルとして自動保存 |
| **繰り返し作業** | 毎回手動 | 一度指示すれば自動で繰り返し |
| **記憶** | 会話中のみ | CLAUDE.mdでルールを永続的に記憶 |

**最大の違い**: Claude.aiはチャット画面に回答を表示します。Claude Codeはフォルダの中に直接入り込み、成果物を作って保存します。

---

## どう動くの？── 実例で見てみましょう

**例1: ドキュメントの一括整理（マーケター／企画担当者）**

```
「Downloadsフォルダにある議事録30件を読んで、
 日付順に整理し、今月の決定事項だけ抽出して。
 月次_決定事項.md として保存して」

→ Claude Codeがやること:
1. フォルダ内の30ファイルをすべて読み込み
2. 日付を特定して並び替え
3. 決定事項のみを抽出
4. 新しいファイルとして自動保存
```

**例2: データ分析（研究者／アナリスト）**

```
「dataフォルダにあるアンケート結果のCSVを分析して、
 主要指標の要約レポートを作って」

Pythonを知らなくても大丈夫。言葉で説明すれば処理してくれます。
```

**例3: コード開発（開発者）**

```
「テストが失敗してるから原因を見つけて直して」
→ テスト実行 → エラー特定 → コード修正 → 再実行
```

## どんな人に向いている？

### こんな方にぴったり

| シーン | 例 |
|------|------|
| **ファイルやドキュメントの繰り返し作業** | 毎週同じ形式のレポート整理、大量のファイル名変更、フォルダ構成の統一 |
| **大量の資料分析** | PDF論文20本の要約、競合サイトの一括収集・整理 |
| **アイデアを形にしたい** | コーディング不要で自分だけのWebサイト、自動化ツール、ダッシュボードを作成 |
| **開発作業** | コード作成、バグ修正、テスト自動化 |

### こんな方にもおすすめ

- 「PythonやExcel関数なしでデータ整理がしたい」
- 「何十件ものドキュメントをいちいち開かず、まとめて処理したい」
- ターミナルに不慣れでも問題ありません ── **デスクトップアプリ**でも使えます

### こんな方にはClaude.ai（Web版）が最適

- 文章作成、翻訳、質問への回答だけが必要な方
- 特定のIDE内だけで作業したい開発者 → CursorやVS Code拡張機能を検討

---

## どこで実行する？

| 環境 | 説明 |
|------|------|
| **デスクトップアプリ** | GUIインターフェース、ターミナル不要 ── **初心者におすすめ** |
| **ターミナル（CLI）** | 最も高機能な標準環境 |
| **VS Code / Cursor 拡張** | エディタ内からそのまま使用 |
| **Webブラウザ** | claude.ai/code からすぐに実行 |
